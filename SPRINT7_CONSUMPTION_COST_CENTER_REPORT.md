# Sprint 7 — Consumption & Cost Center Build Report

Sprint 7 adds internal usage tracking to the Inventory portal. This allows businesses such as laundries, workshops, restaurants, and service operations to track stock consumed internally without incorrectly treating it as sales or waste.

## Lock Result
Sprint 7 is implemented as a ledger-controlled subsystem.

## Key Rule
Consumption reduces inventory only through STORE_USE ledger movements.

## Added
- Cost Centers
- Consumption Templates
- Consumption Events
- Manual Consumption
- Template Execution
- Consumption Dashboard
- Portal UI Summary
- Audit Events

## Deferred
- POS recipe automation
- Service completion triggers
- Stocktake reconciliation
- Forecasting
